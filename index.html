Main Idea

Naive Bayes predicts the class with the highest posterior probability.

Posterior = Prior x Likelihood

For Gaussian Naive Bayes, every numeric feature is treated as a class-specific bell curve.

P(x | class) = 1 / sqrt(2*pi*sigma^2) x e-(x-mu)^2/(2*sigma^2)
